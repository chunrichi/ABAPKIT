# AKIT-HTTP

简单的 `HTTP` 请求的一个封装

> 如果使用可将局部类拷出

**注意** 如果需要使用 HTTPS 需要在 `STRUST` 配置对应的 SSL 证书

## 用途

简单的 REST 请求测试程序

## 使用方法

将 `ZAKIT_DEMO_HTTP` 导入到系统即可, 无需创建屏幕
